# 小诺的错题本

一个面向家长的手机优先错题本原型。

核心流程：

- 拍作业或选择图片
- 先连接家长自己的 AI API
- 批改后整理错题和空白题
- 给出正确答案、解题方法和相似题重做
- 按学科统计阶段变化

## 运行方式

安装 Node.js 后，在项目根目录运行：

```bash
npm start
```

然后打开：

```text
http://localhost:4173/
```

如果 4173 被占用，可以换端口：

```bash
PORT=4174 npm start
```

## 当前可用能力

- 前端可以上传照片并保存到本机浏览器
- 后端提供 `/api/test-ai` 测试用户自己的 API Key
- 后端提供 `/api/review` 调用 AI 批改作业照片
- Gemini / GPT / Claude 支持照片批改
- DeepSeek 当前只支持连接测试；照片批改需要后续先接 OCR

## 部署说明

GitHub Pages 只能托管静态网页，不能运行后端接口，所以只能看界面，不能真实批改。

要让别人真正使用，需要把本仓库部署到能运行 Node.js 的服务，例如 Render、Railway、Fly.io 或自己的服务器，并让用户通过那个服务地址访问。

API Key 由用户自己填写，服务端只作为代理调用，不使用统一后台额度。
